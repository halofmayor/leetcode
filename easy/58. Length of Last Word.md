# 📝 Length of Last Word

| Difficulty | Topics |
| :--- | :--- |
| Easy | String, Iteration |

## 📖 Problem Description
Given a string `s` consisting of words and spaces, return the length of the last word in the string. 

A **word** is a maximal substring consisting of non-space characters only.

---

## 💡 Initial Thoughts & Evolution
My first thought was to split the string into an array, but that could be memory-intensive. I realized that if I clean the trailing spaces first, I can just walk backwards from the end of the string. The moment I hit a space, I know the last word has ended.

> Key Insight: By using `.trim()`, we normalize the string so the last character is guaranteed to be part of the last word.

---

## 🚀 Final Solution

    class Solution {
        public int lengthOfLastWord(String s) {
            // Remove leading and trailing spaces
            s = s.trim();
            
            // Iterate backwards from the last character
            for (int i = s.length() - 1; i > 0; i--) {
                // The first space we find marks the start of the last word
                if (s.charAt(i) == ' ') {
                    return s.length() - i - 1;
                }
            }
            
            // If no space is found, the entire string is one word
            return s.length();
        }
    }

---

## 🛠️ Algorithm
1. Clean Input: Use `s.trim()` to remove any spaces at the beginning or end of the string.
2. Backward Scan: Start a loop from the last index (`s.length() - 1`) moving towards the beginning.
3. Boundary Detection: If a space (`' '`) is encountered, calculate the distance from that space to the end of the string.
4. Calculation: The length is `total_length - current_index - 1`.
5. Fallback: If the loop finishes without finding a space, it means the string consists of only one word; return its full length.

---

## 📊 Complexity Analysis
* Time Complexity: O(n) — In the worst case (one long word), we traverse the entire string once.
* Space Complexity: O(n) — In Java, `trim()` creates a new string object, which can take up to the same amount of space as the original.

## 📝 Code Notes
* Trim Power: The `.trim()` method is essential here because it handles edge cases like `"Hello World  "` where there are multiple spaces at the very end.
* Index Math: The formula `s.length() - i - 1` correctly offsets the 0-based index to give the exact count of characters.
