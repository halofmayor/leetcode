## 📖 Problem Description
Given a string s consisting of words and spaces, return the length of the last word in the string.

A word is a maximal substring consisting of non-space characters only.
---

## 💡 Initial Thoughts & Evolution
I though in converting all to an array and to just see where the last space was and returning the length of the last word. It was not making sense so I tried a different approach of removing the final space, iterating through the end, and when finding an space return the value of the last word length.
---

## 🚀 Final Solution

    class Solution {
    public int lengthOfLastWord(String s) {
        s = s.trim();
        for(int i = s.length() - 1; i > 0; i--){
            if(s.charAt(i) == ' '){
                return s.length() - i - 1;
            }
        }
        return s.length();
    }
}
