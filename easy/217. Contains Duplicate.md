# 📝 Contains Duplicate

| Difficulty | Topics |
| :--- | :--- |
| Easy | Array, Hash Table |

## 📖 Problem Description
Given an integer array `nums`, return `true` if any value appears at least twice in the array, and return `false` if every element is distinct.

---

## 💡 Initial Thoughts & Evolution
I immediately recognized that a Hash-based data structure is the best approach. While a brute-force approach (nested loops) would be O(n²), using a Map allows for O(1) average time complexity for lookups.

> Key Insight: As we iterate, we "record" every number we see. If we encounter a number that is already in our records, we've found a duplicate.

---

## 🚀 Final Solution

    class Solution {
        public boolean containsDuplicate(int[] nums) {
            // Using a HashMap to track seen numbers
            HashMap<Integer, Boolean> dup = new HashMap<>();

            for (int i = 0; i < nums.length; i++) {
                // containsKey runs in O(1) average time
                if (dup.containsKey(nums[i])) {
                    return true;
                }
                dup.put(nums[i], true);
            }
            return false;
        }
    }

---

## 🛠️ Algorithm
1. Initialize HashMap: Create a map to store the integers we encounter.
2. Iterate: Loop through the `nums` array.
3. Check Existence: For each number, check if it already exists as a key in the map.
4. Early Return: If it exists, return `true` immediately.
5. Record: If it doesn't exist, add it to the map and continue.
6. Default Return: If the loop finishes without finding a duplicate, return `false`.

---

## 📊 Complexity Analysis
* Time Complexity: O(n) — We traverse the array at most once.
* Space Complexity: O(n) — In the worst case (no duplicates), we store every element in the HashMap.

## 📝 Code Notes
* Trade-off: We are trading space (O(n) memory) for speed (O(n) time).
* Observation: Surprisingly, HashMap performed faster than HashSet in runtime tests, likely due to how the JVM handles the internal wrapper of the HashSet class.
