# LeetCode Daily Challenges

My main goal is to solve at least **one LeetCode problem every day**.  
This repository contains my solutions, notes, and approaches for each challenge, organized by difficulty and topic.  

## Structure
- **easy/** – Easy problems  
- **medium/** – Medium problems  
- **hard/** – Hard problems  
- **patterns/** – Common algorithms and techniques learned from solving problems  

Each problem has:
- a solution file (Java)
- a markdown file with my thought process, approaches, and lessons learned

---

This repo serves both as a **personal reference** and a way to track my progress over time.
